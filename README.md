# CIS 4340 Lab 02 - GitHub Secure Java Examples

This repository contains 10 short Java examples showing insecure code and corrected code related to secure programming practices in Java.

For each example, I included:
- a vulnerable version of the program
- a fixed version of the program
- a short notes file explaining the problem and the fix

## Folder Structure

Each example folder contains:
- `Vulnerable.java`
- `Fixed.java`
- `notes.md`

## Example List

- `example01` - integer input validation
- `example02` - command-line/file name validation
- `example03` - array index validation
- `example04` - resource handling
- `example05` - null checking
- `example06` - divide-by-zero prevention
- `example07` - protecting internal arrays
- `example08` - correct string comparison
- `example09` - safe loop bounds
- `example10` - command-line argument checking
